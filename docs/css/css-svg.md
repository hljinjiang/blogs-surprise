**svg**

是什么？**可伸缩矢量图形**

放大或改变尺寸的情况下图形质量不会有所损失

作用？    比PNG，JPEG等，尺寸更小，压缩性更强

更伸缩的，放大缩小不会造成图形质量的损失

兼容DOM

在项目中可以使用 svg-sprite-loader 一键配置全局svg的使用

mask遮罩层可以创建svg一样的icon

需要一个背景层background，遮罩层mask
